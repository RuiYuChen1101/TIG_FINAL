Análisis del territorio

En esta práctica, se empleará QGis como una herramienta esencial en la toma de decisiones para acciones de índole territorial. Se buscará identificar áreas idóneas para establecer una reserva para el Lobo Ibérico en la COMUNIDAD DE CASTILLA Y LEÓN. Esto se logrará mediante la combinación lineal de diversas variables geográficas reajustadas, como precipitaciones, temperaturas y elevaciones, además de factores relacionados con la población y la extensión del terreno.

Teledetección

Práctica 5.

Esta práctica implica el uso de QGIS para el análisis de imágenes de satélite. El objetivo es calcular el índice dNBR después de un incendio forestal en Marxuquera (agosto 2008). Se utilizarán imágenes satelitales de Sentinel 2 de la región, tomadas antes y después del incendio. Los datos y mapas de severidad del fuego son cruciales para la elaboración de estrategias de rehabilitación y restauración de emergencia post-incendio. Estos recursos no solo sirven para evaluar la severidad del fuego sobre el terreno, sino también para anticipar posibles consecuencias futuras como inundaciones, deslizamientos de tierra y erosión del suelo.
